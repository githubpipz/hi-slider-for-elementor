Hi-Slider is a custom Elementor widget plugin that adds a Bootstrap-powered carousel slider for posts and images.

==Features: ==

- Easy-to-use Elementor widget
- Fully responsive and mobile-friendly
- Supports images and content
- Lightweight and fast
- Custom styling options

This plugin requires Elementor to be installed and activated.


== Description ==

Hi-Slider is a custom widget for Elementor that adds a responsive post slider using the Bootstrap Carousel. Display your posts, images, or custom content in a sleek, customizable slider.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/hi-slider/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the Elementor editor and drag the "Hi-Slider" widget to your page.

== Frequently Asked Questions ==

= How do I customize the slider? =

You can customize the slider through the Elementor interface. Adjust settings such as the number of posts, animation, and more in the widget's settings.

= Does this work with all themes? =

Yes, as long as the theme supports Elementor, the Hi-Slider widget will work.

= Does this plugin require Elementor? =

Yes, Elementor must be installed and activated for this plugin to work.
